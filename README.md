# Automatic Washing Machine (DLD Project)

## Description

The Automatic Washing Machine is a Digital Logic Design (DLD) project that simulates the operation of an automatic washing machine using digital circuits and logic components. The system controls different stages of the washing process, including water filling, washing, rinsing, and spinning, through predefined logic sequences. The project demonstrates the application of combinational and sequential logic, control systems, state transitions, and automation concepts in a real-world household appliance.

## Features

* Automatic wash cycle control
* Water filling and draining operations
* Washing, rinsing, and spinning stages
* State-based operation sequencing
* Digital circuit implementation using logic gates and flip-flops
* Simulation of real-world appliance automation

## Technologies Used

* Digital Logic Design (DLD)
* Logic Gates
* Flip-Flops
* Counters
* Finite State Machine (FSM) Concepts
* Circuit Simulation Software (Logisim/Proteus/Multisim, if applicable)

## Learning Outcomes

* Design and analysis of digital circuits
* Sequential logic implementation
* Finite State Machine design
* Control system development
* Hardware automation concepts
